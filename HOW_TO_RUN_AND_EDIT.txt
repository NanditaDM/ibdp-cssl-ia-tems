=== DEBUGGING TIPS ===

If Something Doesn't Work:
1. Open browser console:
   - Chrome/Safari: Right-click > Inspect > Console tab
   - OR press Cmd+Option+J (Mac) or F12 (Windows)
2. Look for red error messages
3. The error will tell you which file and line has the problem

Common Errors:
- "Uncaught ReferenceError" = You used a variable that doesn't exist
- "Uncaught TypeError" = You called a method on something that's null/undefined
- "SyntaxError" = You have a typo in your code

To Fix Errors:
1. Note the file name and line number from error
2. Open that file in your editor
3. Go to that line number
4. Fix the issue
5. Save and refresh browser
